<b>Cloud Native Observability</b>

A lightweight observability platform built with a microservices architecture.
The system provides log and metrics ingestion, time-series storage, alerting, and a real-time dashboard for monitoring service health.

<b>Features</b>

Metrics & log ingestion API

Time-series storage for high-volume telemetry

Alerting service with threshold rules

API gateway for unified access

React dashboard for real-time visualization

Containerized microservices (Docker / Kubernetes ready)

<b>Tech Stack</b>

.NET / Node / Go microservices

React + Tailwind dashboard

PostgreSQL / TimescaleDB

Redis / Kafka (optional)

Docker, Kubernetes, CI/CD
